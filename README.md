# homebrew-back-directory

Homebrew tap for [back-directory](https://github.com/01-mu/back-directory).

## Install

```sh
brew tap 01-mu/back-directory
brew install back-directory
```

After installation, add the matching wrapper to your shell rc:

```sh
# bash
source "$(brew --prefix)/opt/back-directory/share/back-directory/bd.bash"

# zsh
source "$(brew --prefix)/opt/back-directory/share/back-directory/bd.zsh"
```
